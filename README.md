# Controle Patrimonial

Sistema de controle patrimonial desenvolvido com:

## Backend
- Node.js
- Express
- Prisma
- PostgreSQL
- JWT

## Frontend
- React
- Vite
- Axios
- Bootstrap

## Funcionalidades
- Login
- Cadastro de patrimônios
- Listagem de patrimônios
- Exclusão de patrimônios

## Como executar

### Backend

npm install
npm run dev

### Frontend

npm install
npm run dev
